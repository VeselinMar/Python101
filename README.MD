## How to Run

To run any of the included scripts:

1. Make sure you have **Python 3.x** installed.
2. Clone the repository:

```bash
git clone https://github.com/VeselinMar/Python101.git
cd Python101
source .venv/bin/activate
```
3. Run a script using Python:
```bash
python game.py
```
